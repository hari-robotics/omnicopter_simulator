# Omnicopter Simulator
This repository is for simulating the omnicopter with designed control theory, compiled with __ROS2__ and __Gazebo Classic__.

## Pre-requirements
* ROS2 Humble
* Gazebo Classic
* Gazebo ROS2 Control Plugin
* Eigen3
* OSQP
* OSQP-Eigen

## Installation
```bash
mkdir omnicopter_ws
cd omnicopter_ws
git clone https://github.com/UoN-Hari/omnicopter_simulator.git src
```
